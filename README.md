# 中文大模型能力评测榜单（持续更新）
- 目前已囊括11个大模型，覆盖百度文心一言、chatgpt、阿里通义千问、讯飞星火等商用模型，以及belle、chatglm6b 等开源大模型。
- 支持多维度能力评测，包括分类能力、信息抽取能力、阅读理解能力。
- 不仅提供能力评分排行榜，也提供所有模型的原始输出结果！
